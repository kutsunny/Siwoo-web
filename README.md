# 지오메트리 대시 — 유명 레벨 가이드

지오메트리 대시(Geometry Dash) 유명 레벨들의 **역사 · 데몬리스트 순위 변동 · 베리파이(인증) 과정**을 정리한 한국어 가이드 사이트입니다.
순수 HTML/CSS/JS로 만들어, 설치 없이 `index.html`을 브라우저로 열면 바로 동작합니다.

## 실행 방법

별도 빌드/서버가 필요 없습니다.

- 파일 탐색기에서 `index.html` 더블클릭, 또는
- WSL 환경: `explorer.exe index.html` 로 윈도우 브라우저에서 열기

> 변경 후 화면이 안 바뀌면 강력 새로고침: **Ctrl + Shift + R**

## 폴더 구조

```
Siwoo-web/
├── index.html      메인 페이지 (히어로 / 게임 소개 / 개념 / 레벨 타임라인 / 모달)
├── README.md       이 문서
├── css/
│   └── style.css   네온 다크 테마 + 레이아웃 + 필터/카드/모달 스타일
└── js/
    ├── data.js     레벨 데이터 배열(LEVELS) + 카테고리 목록(CATEGORIES)
    └── main.js     카드/필터 렌더링, 모달 제어, 일러스트(SVG) 생성기
```

## 주요 기능

- **레벨 타임라인** — 21개 유명 레벨을 시간순(2015 → 2026)으로 카드 표시
- **카테고리 필터** — `1위 경쟁 / 장르 개척 / 혁신 / 역사 / 불가능 레벨` 버튼(개수 표시)
- **상세 모달** — 카드 클릭 시 제작자·베리파이어·연도·최고순위 + 역사 + (있으면) 논란
- **논란 섹션** — 사실 단정 대신 "커뮤니티 의견"으로 구분 표기, 카드에 `논란` 배지
- **이름 맞춤 일러스트** — 레벨별 테마 색 + 모티프로 SVG를 코드 생성(시드 기반이라 항상 동일)
- 모바일 반응형, 키보드 접근성(Enter/Space로 카드 열기, ESC로 모달 닫기)

## 레벨 추가하는 법

`js/data.js`의 `LEVELS` 배열에 객체를 하나 추가하면 카드가 자동 생성됩니다(파일 상단 주석에 필드 설명 있음).

```js
{
  id: "고유-id",            // 영문, 공백 없이
  name: "레벨 이름",
  creator: "제작자 / 호스트",
  verifier: "베리파이어",
  year: 2024,               // 모르면 "—"
  peakRank: "1위",          // 또는 "톱 5", "비공식" 등
  category: "1위 경쟁",     // CATEGORIES 중 하나
  tag: "짧은 수식어",
  motif: "fire",            // 아래 모티프 목록 참고
  colors: ["#메인", "#보조"],
  // image: "사진주소.jpg", // (선택) 있으면 SVG 대신 이 이미지 사용
  summary: "카드 한 줄 요약",
  history: "문단1\n\n문단2",  // 빈 줄(\n\n)로 문단 구분
  // controversy: "논란 내용", // (선택) 있을 때만 모달에 논란 박스 표시
}
```

### 일러스트 모티프 (`motif`)

`js/main.js`의 `MOTIFS` 객체에 정의되어 있습니다.

`circles`(동심원) · `ice`(얼음 결정) · `blood`(핏방울) · `fire`(불꽃) ·
`wave`(지그재그 웨이브) · `tidal`(큰 파도) · `pulse`(맥박) · `stars`(별/우주) ·
`gold`(금화) · `key`(열쇠) · `gem`(보석) · `orbit`(행성 궤도) · `sound`(이퀄라이저) ·
`spikes`(기본: 큐브+가시)

새 모티프가 필요하면 `MOTIFS`에 `이름: function(c1, c2, rng) { return "<svg 내부 마크업>"; }` 형태로 추가하세요.

## 데이터 정확성 메모

- 레벨 사실관계(제작자·베리파이어·연도)는 공개 위키(geometrydash.wiki.gg, fandom 등)를 참고했습니다.
- 데몬리스트 순위는 계속 변동되므로 `peakRank`는 "최고/대표 위상" 기준입니다.
- 민감한 부분(핵 베리파이 등)은 초창기 업로드 관행으로 맥락을 설명하고, 논란은 "커뮤니티 의견"으로 구분했습니다.
- 주의: **Sakupen Hell**(불가능 레벨)과 **Sakupen Circles**(나인 서클즈류)는 서로 다른 레벨입니다.
